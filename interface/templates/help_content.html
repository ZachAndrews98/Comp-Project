Images

There are several functions on the Images tab which are explained in detail below

Run Image

Runs a specified image with any optional arguments the user supplies

Delete Image

Deletes a specified image, can not be in use by a container

Pull Image

Download a specified image from Docker Hub

Push Image

Upload an image to Docker Hub, requires that the user has an account

Generate Dockerfile

Generate a Dockerfile within a specified directory automatically

Build Image

Build an image based on a path to a Dockerfile and a specified name
